# Quiz Management System

A Django-based web application for managing quizzes, users, and scores.  
This project provides both a **quiz portal** for participants and an **admin dashboard** for organizers.

## ✨ Features
- 🔐 Secure user authentication (login/logout)
- 📝 Quiz participation with automatic scoring
- 📊 Results page showing performance insights
- 🛠 Admin dashboard with:
  - Manage Users and Groups
  - Add/Edit Questions
  - Track Scores
- 🎨 Custom branding: “Quiz Management System” header and portal

## Tech Stack
- Django (Python framework)
- SQLite (default database)
- HTML templates (quiz, result, login, portal)
- CSS for styling
